# Battleship-project

## 📌 Prerequisites
Ensure you have the following installed:
- **Node.js** (v14+ recommended) → [Download here](https://nodejs.org/)
- **npm** (comes with Node.js) or **yarn** (alternative package manager)

## 📥 Installation & Setup
1. **Clone the repository**:
   ```sh
   git clone <your-repo-url>
   cd <your-project-folder>

2. **Install dependencies**:
   ```sh
   npm install
   ```
## 🚀 Running the Development Server
To start the Vite development server, run:
```sh
npm run dev
```
```
VITE v4.0  ready in 500ms

  ➜  Local:   http://localhost:5173/
```
Open the URL in your browser to see the app running.

---

## 🛠️ Building for Production

To generate an optimized production build, run:
```sh
npm run build
```
or
```sh
yarn build
```
This will create a `dist/` folder containing the **final optimized production files**.

---

## ✅ Preview Production Build

To preview the production build locally, run:
```sh
npm run preview
```
or
```sh
yarn preview
```
This will serve the built project at a local URL.

---

## 🔥 Useful Commands

| Command             | Description |
|---------------------|-------------|
| `npm install`      | Install dependencies |
| `npm run dev`      | Start development server |
| `npm run build`    | Build for production |
| `npm run preview`  | Preview production build |
